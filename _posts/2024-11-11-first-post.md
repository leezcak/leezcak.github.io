1. **나는 어떤 히스토리를 가지고 데이터 분석트랙에 참여하게 되었나요?**
    - 통계학과를 졸업하고 데이터 분석/엔지니어링을 더 심도 있게 배우고 싶어서 신청했습니다.
2. **본 코스 수료 후, 어떤 데이터 분석가 혹은 원하는 직무로 성장하고 싶나요?**
    - 금융계에 관심이 많아 financial analyst로 취업하는 것이 단기적 목표입니다.
3. **그 외에 내일배움캠프에 기대하는 것이 있다면 자유롭게 작성해 주세요.**
    - 저는 이미 SQL, Python, Tableau 등 경험이 꽤 있는 편이라 데이터 파이프라인 같은 심화과정들도 많이 배울 수 있는 기회가 있었으면 좋겠습니다.


1. **내가 데이터 분석 트랙에 참여한 계기는 무엇인가요?**
    - 통계학과를 졸업하고 데이터 분석/엔지니어링을 더 심도 있게 배우고 싶어서 신청했습니다.
2. 내가 이해한 **데이터 분석가(혹은 생각했던 직무)**는 어떤 역할을 하는 사람인가요? 
    - 데이터를 관리, 추출, 분석, 시각화, 보수하는 역할을 맡는 사람.
3. 데이터 분석을 경험해보셨나요? 해보셨다면 어떤 경험을 하셨는지를 작성해주시고,
아니라면 데이터 분석에 대해 찾아본 것을 작성해주세요.
    - 작년에 데이터 분석 써머인턴을 했습니다. 외부 데이터를 갖고와 내사 데이터베이스에 연결해서 시각화하거나, 데이터 파이프라인을 통해 외부 데이터를 내사 데이터베이스로 연결하는 일을 했습니다.
4. **데이터 분석가의 역할을 수행하는 데에 있어 나의 강점과 연관된 부분은 무엇이라고 생각하나요?** 
혹은 보완, 개선하고 싶은 개인 역량이 있나요 ?
    - 강점: 학부 백그라운드와 다양한 데이터 관련 경험
    - 개선점: 데이터 분석과 관련된 심화 데이터 업무
5. **본 코스 수료 후, 어떤 데이터 분석가(혹은 생각했던 직무)로 성장하고 싶나요?**
    - 단기적으로는 금융기관에서 financial analyst로 일하고 싶습니다.
6. **본 코스 수료 후, 데이터 분석가(혹은 생각했던 직무)가 된 5년 후 목표로하는 점은 무엇인가요?**
    - 유학시절 구글 런던 지사 앞을 지나가며 이 세상에서 제일 데이터가 많은 회사에 한 번쯤은 일해보고 싶다는 생각을 했습니다. 원래 영국에서 취업을 목표했지만 아쉽게도 비자 문제 때문에 귀국하게 된지라 한국에서 몇 년 정도 경험을 쌓고 해외 기업으로 진출하고 싶습니다.
